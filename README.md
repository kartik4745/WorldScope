# WorldScope

WorldScope is a responsive web application that allows users to explore countries using real-time data from a public API. Users can search, filter, and sort countries dynamically through an interactive interface.

---

## Features

* Search countries by name
* Filter countries by region
* Sort countries by population (ascending and descending)
* Loading indicator while fetching data
* Displays message when no results are found
* Dark mode toggle
* Debounced search for better performance
* Responsive design for all screen sizes

---

## Tech Stack

* HTML5
* CSS3
* JavaScript (ES6)
* REST Countries API
